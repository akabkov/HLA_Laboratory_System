Справка по лицензированию Qt / PySide6
======================================

Приложение использует PySide6 (Qt for Python).

Проверенная среда сборки проекта на 2026-04-23:
- PySide6 6.11.0
- PySide6_Essentials 6.11.0
- PySide6_Addons 6.11.0
- shiboken6 6.11.0

Метаданные установленных wheel-файлов для этих пакетов указывают:
- LGPL-3.0-only OR GPL-2.0-only OR GPL-3.0-only

Для данного проекта предполагается распространение Qt for Python по маршруту
LGPLv3, если для конкретного релиза не получена и не применяется отдельная
коммерческая лицензия Qt.

При подготовке финального дистрибутива необходимо:
1. включить применимые тексты открытых лицензий Qt, на которые опирается
   релиз;
2. включить точные license/notice-файлы, поставляемые с тем wheel-файлом
   PySide6, который использован для сборки релиза;
3. включить файл THIRD_PARTY_LICENSES.txt данного проекта;
4. сохранить заменяемость библиотек Qt/PySide6 в поставляемом bundle и не
   препятствовать отладке изменений в соответствующих LGPL-компонентах, если
   релиз не распространяется на основании отдельной коммерческой лицензии Qt;
5. сверить фактическое содержимое папки dist/HLA_Laboratory_System/ с набором
   Qt/PySide6-модулей, указанным в уведомлениях.

Причина включения этой справки:
документация Qt for Python указывает, что Qt for Python содержит код не только
под LGPL или коммерческими условиями Qt, но и под отдельными лицензиями
первоначальных авторов. Qt рекомендует программам, использующим Qt for Python,
указывать такие компоненты и приводить применимые лицензионные уведомления в
документации.

Практический порядок:
после установки PySide6 в среде сборки необходимо проверить содержимое
установленных пакетов и перенести в релизный комплект license/notice-файлы,
соответствующие именно этой версии PySide6. Рекомендуемая сборка проекта
использует PyInstaller --onedir, что удобнее для сохранения заменяемости
LGPL-компонентов. Если в релизе изменяются файлы Qt/PySide6/shiboken, перед
распространением необходимо отдельно проверить обязанности по предоставлению
исходного кода соответствующих компонентов.

В данный репозиторий включены:
- licenses/LGPL-3.0.txt
- licenses/GPL-2.0.txt
- licenses/GPL-3.0.txt
- licenses/Qt-Commercial-Reference.txt

Файл Qt-Commercial-Reference.txt взят из метаданных установленного wheel-файла
как upstream-справка. Его наличие само по себе не означает, что проект
распространяется по коммерческой лицензии Qt.
