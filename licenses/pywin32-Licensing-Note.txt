Справка по лицензированию pywin32
=================================

Проект использует pywin32 для Windows COM-интеграции при преобразовании
файлов заключений Word (`.doc` / `.docx`) в PDF во время импорта.

Проверенная среда сборки проекта на 2026-04-23:
- pywin32 311

Публичные метаданные pywin32 указывают:
- PyPI project page: https://pypi.org/project/pywin32/
- declared license: Python Software Foundation License (PSF)
- upstream source repository: https://github.com/mhammond/pywin32

Установленный wheel также содержит отдельные BSD-style уведомления для
компонентов win32 и win32com, используемых данным проектом. Эти уведомления
включены в файл:
- licenses/pywin32-LICENSE.txt

Перед подготовкой распространяемого релизного комплекта необходимо проверить,
что файл licenses/pywin32-LICENSE.txt соответствует именно тому wheel-файлу
pywin32, который использован в сборке. Если версия pywin32 или источник
пакета изменились, лицензионный файл и THIRD_PARTY_LICENSES.txt нужно
обновить.

Microsoft Word не распространяется в составе данного приложения. Его установка
и использование регулируются отдельными условиями Microsoft.
