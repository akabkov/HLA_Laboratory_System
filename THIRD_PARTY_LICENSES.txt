Build-time tools and generated bundle
-------------------------------------

PyInstaller 6.19.0 is used to create the Windows executable bundle.
PyInstaller itself is a build tool, but its bootloader and runtime hooks may be
embedded into the generated executable/bundle.

Installed PyInstaller metadata declares:
- GPLv2-or-later with a special exception which allows use of PyInstaller to
  build and distribute non-free programs, including commercial programs.

Bundle with distribution:
- licenses/PyInstaller-COPYING.txt

Related build helper packages installed in the build environment, such as
altgraph, packaging, pefile, pyinstaller-hooks-contrib, pywin32-ctypes and
setuptools, are not intended to be redistributed as separate runtime
components of the application bundle. If you redistribute the build environment
or any of those packages themselves, review and include their separate license
terms and notices.
